What is DOM and what is the purpose of DOM?

DOM full meaning is Document Object Model. it is interface for web documents. dom is programming interface for document. the web programs can change the document structure, style, and content.

How will you select HTML elements using DOM? Name the DOM methods.

DOM use 5 HTML ELEMENTS for select. the 5 element is getElementsByTagName(), getElementsByClassName(), getElementById(), querySelector(), querySelectorAll()

What is event bubble?

The Event bubbling in JavaScript is a type of event propagation. The event triggers the innermost target element and consecutively triggers the parent element of the target element in the same hierarchy until it triggers the outermost element.

What is a callback function and why will you use it?

A callback is a function that is passed as an argument to another function, and is called after the main function has finished its execution. The main function is called with a callback function as its argument, and when the main function is finished, it calls the callback function to provide a result.
